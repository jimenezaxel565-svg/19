Problema común

Componente que hace todo

// ❌ 300 líneas, 5 responsabilidades
function Dashboard() {
  const [user, setUser] = useState(null);
  const [posts, setPosts] = useState([]);

  useEffect(() => {
    fetchUser().then(setUser);
    fetchPosts().then(setPosts);
  }, []);

  const formatDate = (d) => new Date(d).toLocaleDateString();
  // ... más lógica, filtros, etc.

  return ( <div>UI enorme</div> );
}

Consecuencias: difícil de testear, cualquier cambio rompe todo.

Solución

Divide en Custom Hooks

// ✅ Cada hook, una responsabilidad
function useUser() {
  const [user, setUser] = useState(null);
  useEffect(() => { fetchUser().then(setUser); }, []);
  return user;
}

function usePosts() {
  const [posts, setPosts] = useState([]);
  useEffect(() => { fetchPosts().then(setPosts); }, []);
  return posts;
}

// Componente solo orquesta
function Dashboard() {
  const user = useUser();
  const posts = usePosts();
  return <DashboardView user={user} posts={posts} />;
}

Resultado: código reusable, testeable y legible.
